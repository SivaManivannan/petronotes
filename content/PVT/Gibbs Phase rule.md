# Gibbs Phase rule

Gibbs phase rule states that the number of intensive variables (i.e. degrees of freedom), F, that must be specified to determine the thermodynamic state of equilibrium for a mixture containing $n$ components distributed in $P$ phases (gas, liquid, solid), is
$$F = n-P+2$$

**Intensive (thermodynamic) variables** do not depend on the amount of material in the system, e.g. **temperature, pressure and density**.
Extensive variables depend on the extend of the system, e.g. flow rate, total mass, liquid volume.

# Examples

- For a two-phase single component system ($n=1$ and $P=2$ ) with $F=1$, only pressure or temperature needs to be specified to determine the thermodynamic state of the system
- For a two-phase two component system ($n=2$ and $P=2$ ) with $F=1$, both pressure and temperature needs to be specified to determine the thermodynamic state of the system

> [!tip] When other forces like gravity are considered, for example when computing compositional variation with depth, the phase rule should be modified as $F=n-P+3$
